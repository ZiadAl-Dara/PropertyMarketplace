PROJECT TITLE: London Property Marketplace
PURPOSE OF PROJECT: To create a program that loads the AirBnB London data set from the provided csv file into a Java data structure and inputs that into a GUI that lets the user search for different properties.
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: Create a new GUI()
AUTHORS: Ziad Al-Dara, Mihnea-Andrei Radulescu, Alexander Bass, William Cohen
USER INSTRUCTIONS: Use the BlueJ IDE to boot up the application.
